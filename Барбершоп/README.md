##  Обзор проекта

Система моделирует ключевые бизнес-процессы барбершопа:

- Учёт **товаров и услуг** по партиям и срокам годности  
- Контроль **остатков** и **себестоимости**  
- Расчёт **выручки** и **прибыли**  
- **Регламентированный учёт** по РСБУ  
- Начисление **зарплаты** (оклад, премии, больничные)
- **Авторизация пользователей** в системе и хранение информации об авторе документа  
- Добавлены подсистемы Организация, Префиксация объектов, Валюты, Печать, Дополнительные отчеты и обработки из БСП
- **Монитор руководителя** отслеживание показателей бизнеса, отслеживание новостей и курса валют с помощью связки с веб-сервисами по API.

##  Ключевые особенности

- Внедрение библиотеки стандартных подсистем (БСП) с постановкой на поддержку поставщика.
- Единая стилистика интерфейса (заданные цвета, логотип, заставка, картинки для подсистем).
- Иерархические справочники (Номенклатура, Контрагенты).
- Функциональная опция «Использовать склады» для управления видимостью складских объектов.
- Использование механизмов БСП (методы СообщитьПользователю, ЗначениеРеквизитаОбъекта и др.).
- Автоматическое заполнение реквизитов (ответственный, дата и номер документа и т. д.).
- Расчёт сумм в табличных частях документов с выводом итогов в подвале формы.
- Внешняя печатная форма «Товарная накладная» для документа «Реализация» с подключением через механизмы БСП.
- Полноценная подсистема бухгалтерского учёта с планом счетов, субконто, регистром бухгалтерии и проводками.
- Автоматизированная загрузка прайс‑листов из файлов Excel.
- Валидация данных (проверка даты записи, длины ИНН, типов счетов учёта и т. д.).
- Гибкая настройка видимости объектов в зависимости от ролей пользователя (Авторизованный/Неавторизованный).

##  Выполненные задачи
- Внедрение обязательных и дополнительных подсистем БСП:
- Базовая функциональность, Обновление версий ИБ, Пользователи, Организации, Печать и др.
- Настройка пользовательского интерфейса:
- Разработка и установка стиля оформления.
- Загрузка картинок для подсистем, логотипа и заставки.
- Создание подсистем: Продажи, Покупки, Управление персоналом, Денежные средства, Справочники, Сервис и настройки.
- Разработка перечислений (Типы организаций, Типы номенклатуры и др.) и групп команд.
- Создание справочников (Номенклатура, Сотрудники, Должности, Контрагенты, Договоры, Организации и др.) с формами элементов и списков.
- Разработка документов (Поступление товаров и услуг, предварительная запись, реализация, приходный кассовый ордер, расходный кассовый ордер).
- Создание регистров накопления (Товары на складах, Заказы клиентов и др.) и регистра сведений (Цены номенклатуры).
- Разработка аналитических отчётов (Товары на складах, Продажи и др.).
- Разработка внешней печатной формы «Товарная накладная»:
- Подключение через механизмы БСП.
- Вывод реквизитов шапки, табличной части, итогов и подвала (в т. ч. сумм прописью).
- Создание подсистемы бухгалтерского учёта:
- План видов характеристик «Виды субконто хозрасчётные».
- План счетов «Хозрасчётный» с предопределёнными счетами (10, 41, 50, 60 и др.).
- Регистр бухгалтерии «Хозрасчётный» с корреспонденцией счетов.
- Движения по регистру от документов (Поступление товаров и услуг, Реализация и др.).
- Доработка вспомогательных объектов:
- Добавление реквизита «Счёт бухгалтерского учёта» в справочник Номенклатура с ограничениями по типам номенклатуры.
- Разработка журнала документов «Журнал хозяйственных операций» и отчёта «Оборотно‑сальдовая ведомость по счёту».
- Реализация механизма загрузки прайс‑листов.
- Форма загрузки с указанием даты и вида цены.
- Обработка ошибок и вывод сообщений о результате загрузки.
- Настройка командного интерфейса: добавление группы команд «Бухгалтерский учёт».
