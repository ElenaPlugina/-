##  Обзор проекта

Система моделирует ключевые бизнес-процессы барбершопа:

- Учёт **товаров и услуг** по партиям и срокам годности  
- Контроль **остатков** и **себестоимости**  
- Расчёт **выручки** и **прибыли**  
- **Регламентированный учёт** по РСБУ  
- Начисление **зарплаты** (оклад, премии, больничные)
- **Авторизация пользователей** в системе и хранение информации об авторе документа  
- **Мобильная запись клиентов** с аутентификацией и синхронизацией через REST
- Добавлены подсистемы Организация, Префиксация объектов, Валюты, Печать, Дополнительные отчеты и обработки из БСП
- **Монитор руководителя** отслеживание показателей бизнеса, отслеживание новостей и курса валют с помощью связки с веб-сервисами по API.

##  Техническая архитектура

| Компонент | Описание |
|:-----------|:----------|
| **Платформа** | 1C:Предприятие **8.3.24+** (управляемые формы, СКД 2.0, мобильная платформа 8.3.22+) |
| **СУБД** | Файловая (локально) / **MSSQL 2019+** для серверного режима (`/SQL/`) |
| **Язык** | Встроенный 1C (модули объектов и форм), язык запросов 1С, виртуальные таблицы |
| **Мобильное приложение** | 1C Мобильная платформа, эмуляция в Android Studio, сборка **APK** |

##  Ключевые особенности

- Полноценная модель внутреннего учета и аналитики

- Расширяемая структура конфигурации  
- REST-интерфейс для интеграции с мобильными клиентами  
- Поддержка офлайн-режима на мобильной платформе
