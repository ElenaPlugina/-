<h1 align="center"> Салон красоты</h1>
<p align="center">
  <b>Комплексная нетиповая 1C-конфигурация для автоматизации салона красоты</b><br>
</p>

![GitHub](https://img.shields.io/badge/github-%23121011.svg?style=for-the-badge&logo=github&logoColor=white)

*Проект автоматизации салона красоты на платформе 1С:Предприятие. Реализованы оперативный и бухгалтерский учёт, отчётность, печатные формы, загрузка прайс‑листов из Excel, а также мобильная конфигурация для онлайн‑записи и HTTP‑интеграция с основной базой.*

**В MVP — ядро учёта:** справочники, документы, регистры накопления и сведений, отчёты на СКД, внедрение БСП и фирменный интерфейс. Во второй части — бухгалтерский учёт (план счетов, субконто, регистр бухгалтерии, проводки, ОСВ), внешняя печатная форма товарной накладной, загрузка Excel, мобильная запись с авторизацией и отправкой данных.

**Ключевые механизмы:** контроль отрицательных остатков, функциональная опция для динамической видимости, валидация данных, разделение ролей, обработка ошибок.

Проект показывает навыки разработки конфигурации «с нуля», работы с БСП и СКД, ведения бухгалтерского учёта, построения интеграций и внимания к пользовательскому опыту.
