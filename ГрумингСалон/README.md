##  Обзор проекта

Система моделирует ключевые бизнес-процессы груминг-салона:

- Учёт **услуг, материалов и товаров** по партиям и срокам годности  
- Учёт **животных** и истории их обслуживания  
- Контроль **остатков**, **себестоимости** и **прибыли**  
- Расчёт **зарплаты** (оклад, премии, больничные, сдельная оплата)  
- Ведение **регламентированного учёта** по РСБУ  
- **Авторизация пользователей** и хранение информации об авторе документа  
- **Онлайн-запись клиентов и питомцев** с синхронизацией через REST  
- **Аналитика** по мастерам, видам услуг и породам животных  
- **Монитор руководителя** отслеживание показателей бизнеса, отслеживание новостей и курса валют с помощью связки с веб-сервисами по API.

| Компонент | Описание |
|:-----------|:----------|
| **Платформа** | 1C:Предприятие **8.3.24+** (управляемые формы, СКД 2.0, мобильная платформа 8.3.22+) |
| **СУБД** | Файловая (локально) / **MSSQL 2019+** для серверного режима (`/SQL/`) |
| **Язык** | Встроенный 1C (модули объектов и форм), язык запросов 1С |
| **Мобильное приложение** | 1C Мобильная платформа, эмуляция в Android Studio, сборка **APK** |

##  Ключевые особенности

- Полный цикл учёта: услуги, товары, клиенты, животные, финансы  
- Единая база данных для фронта и бэк-офиса  
- Расширяемая архитектура с возможностью внедрения CRM-функций  
- REST-интерфейс для мобильных клиентов и веб-записи  
- Гибкая настройка графиков работы мастеров  
- Поддержка офлайн-режима и синхронизации при подключении
